# JWT Authentication Project

پروژه احراز هویت JWT با FastAPI و Angular

## 🚀 فناوری‌ها

- **Backend:** FastAPI, Python 3.12, SQLAlchemy, JWT
- **Frontend:** Angular 20, TypeScript, Signals
- **Database:** SQLite
- **Container:** Docker, Docker Compose
- **CI/CD:** GitHub Actions
